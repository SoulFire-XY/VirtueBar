How to Code in VirtueBar?

This might be a big question for you that "how to code in VirtueBar? " or "where to start learning? " .
You have come to the right place. The Official Wiki of VirtueBar handwritten by the developer and designer of the language.
Now without wasting our time, lets get started.

Insight about VirtueBar:

VirtueBar is a simple, East-to-learn, high level programming language developed by a single person in python and bare code from scratch. It is in development and many features are yet to be added.

#Grammer ########################################################################################################################################################################

Grammer:

Expression
=> KEYWORD:VAR IDENTIFIER Equals Expression
=> Comparison-Expression ((KEYWORD:[WITH|OR] Comparison-Expression) *Optional


Comparison Expression
=> Arithmetic-Expression ((DE|LT|GT|LTE|GTE|NT) Arithmetic-Expression) *Optional
=> INVER Comparison-Expression (NOT operations)


Arithmatic Expression
=> term ((PLUS | MINUS) term) *Optional


Term
=> factor ((MUL | DIV) factor) *Optional


Factor
=> (PLUS | MINUS) factor
Power


Power
=> atom (POWER factor) *Optional


Atom
=> INT|FLOAT|IDENTIFIER
LPAREN Expression RPAREN


If Expression
=> KEYWORD:IF Expression KEYWORD:ACT Expression
=> (KEYWORD:ELIF Expression KEYWORD:ACT Expression) *Optional and Unlimited
=> (KEYWORD:OTHER Expression) *Optional


For Loop Expression
=> KEYWORD:FOR IDENTIFIER Equals Expression KEYWORD:TO Expression KEYWORD:ACT Expression
=>KEYWORD:FOR IDENTIFIER Equals Expression KEYWORD:TO Expression (KEYWORD:STEP Expression) KEYWORD:ACT Expression


While Loop Expression
=> KEYWORD:WHILE Expression KEYWORD:ACT Expression
